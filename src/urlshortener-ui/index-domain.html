URL Shortener

Configure Custom Domain

Specify the fully-qualified custom domain for URL Shortener. The value will be used as base url part of shortened URL produced by URL Shortener service. In order for shortened URL redirection to work, It should usually be configured correctly at your DNS service provider and integrated with this URL Shortener service backend. Example syntax may look like (no quote): 'https://h919hqbhel.execute-api.ap-southeast-1.amazonaws.com/njdev/NJRedirect-URL'
